==============JS==============
-> Arrow Function
- Arrow functions were introduced in ES6.
- Arrow functions provide you with an alternative way to write a shorter syntax compared to the function expression.
- Arrow function share surrounding/lexical this keyword.

-> Destructuring
- Used to destruct the data structure like array or object.
ex. const [name, age] = ['mark', 28]

->Arrays method in ES6
- indexof() - Get the index of the first element in the array.
- find() - Get the value of the first element in the array
- from() - It converts array-like values and iterable values into arrays.

-> for/in - loops through the properties of an object.
- Iterates over the enumerable properties of an object, in an arbitrary order.

-> for/of - loops through the values of an iterable object.
- Iterates over values that the iterable object defines to be iterated over.
