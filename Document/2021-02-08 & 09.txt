=>AJAX
- AJAX = Asynchronous JavaScript and XML.
- AJAX is a technique for creating fast and dynamic web pages.
- AJAX allows web pages to be updated asynchronously by exchanging small amounts of data with the server behind the scenes.
- XMLHttpRequest object can be used to exchange data with a web server.
- open() and send() methods of the XMLHttpRequest object use to send a request to a server.
- responseText property returns the server response as a JavaScript string.
